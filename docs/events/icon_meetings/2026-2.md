### Dominik Schumacher
No update.

### Victoria Bauer
Ran an ICON simulation forced by CESM output; in contact with Cosimo. The run covering 1980–2024 is complete; results have not yet been analyzed.

### Andrea Stenke
Running ICON-ART with interactive chemistry on Euler. Switching from the C2SM repository to ICON-NWP took some time, but the issue is now resolved. Currently dealing with chemistry-related issues.

### Sylvaine Ferrachat
Recoupling HAM to the latest ICON version. Users in the Lohmann group are working on various topics. Running ICON-HAM at higher resolutions (40 km, 20 km, 10 km).

### Matthias Kraushaar
Working on a JupyterHub setup that allows the use of a single GPU. One of the first users will be a course with Oliver Fuhrer next week.

### Will Sawyer
CI for ICON-merge is still missing. A follow-up project is planned to interface with ComIn, in collaboration with DKRZ.

### Christoph Müller
Working on ICON-merge. Also working on EMVORADO; MCH wants to run it in production. Working with FOF files, as EMVORADO FOF files have some special properties. Trying to rewrite probtest.

### Christian Steger
Working on a parameterization for precipitation forcing; quite advanced and almost at merge request (MR) state. A subgrid-scale scheme will soon become available.

Master's students:
- Working on TERRA-URB for numerical weather prediction; will look into land cover (too little urban, too much forest).
- Investigating precipitation bias in southerly flow situations. Also tried the two-moment microphysics scheme; it improves results in some cases but not in others.

### David Leutwyler
Fixed a small issue in the clipping of tracers and hydrometeors; was merged last week. Spent a lot of time taking over the precipitation issue.

### Alina Yapparova
Update on behalf of Xavier: a new release with a tag is available. Another release including EMVORADO is planned.

Working on data assimilation with the EMVORADO forward operator; a merge request on EMVORADO was just approved.
